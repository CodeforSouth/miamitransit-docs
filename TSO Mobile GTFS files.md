TSO Mobile GTFS files
=====================

TSO Mobile provides GTFS feeds for each public transit agency they serve.

Base URL
--------

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
http://transit.tsomobile.com/<agency id>/GoogleTransit.zip
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
